# react-native-rounded-progress-bar-ios

## Getting started

`$ yarn add react-native-rounded-progress-bar-ios`

## Usage
```javascript
import RoundedProgressBar from 'react-native-rounded-progress-bar-ios';

<RoundedProgressBar	
	percent={99.9}	
	borderWidth={4}	
	size={40}	
	color="#F02D00">	
	<Text>Loading...</Text>
</RoundedProgressBar>
```
